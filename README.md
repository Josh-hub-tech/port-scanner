A simple Python port scanner to detect open ports on a target host.

## 🚀 Usage

```bash
# Scan common ports (1-1024)
python3 port_scanner.py example.com

# Scan a custom range
python3 port_scanner.py 192.168.1.1 -p 20-100
```
